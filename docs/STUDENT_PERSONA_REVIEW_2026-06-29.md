# 학생 페르소나 기반 커리큘럼 리뷰 - 2026-06-29

## 목적

이번 리뷰는 학생 입장에서 전공, 직무, 교육 내용이 충분히 연결되는지 확인하기 위한 점검이다. 단순히 자료 개수가 많은지가 아니라, 학생이 선택한 세부 직무에서 다음 질문에 답할 수 있는지를 기준으로 봤다.

- 내 전공에서 왜 이 직무로 이어지는지 이해되는가
- 부족 역량을 체크하면 로드맵과 교육 추천이 좁혀지는가
- 필수 Starter Pack이 먼저 보이는가
- 영어/한국어 자료가 구분되고, 영어 자료도 볼 단원/섹션이 지정되는가
- 주차별 과제가 산출물로 남는가

## 발견 및 조치

1. P5 제조 AI 확장 학생은 추천 자료는 충분했지만 Starter Pack이 비어 있었다.
   - 조치: `Google Machine Learning Crash Course`, `Machine Learning Onramp`를 Starter Pack으로 지정했다.

2. P6 데이터센터 전력 학생은 Schneider `Data Center University`가 상위 추천에 있었지만 본문 GET 기준 403으로 확인되어 학생용 핵심 자료로 부적합했다.
   - 조치: Schneider 자료를 학생 노출에서 제외하고 후보 CSV에만 남겼다. 대신 KDCC `데이터센터 구축·운영 제도 연구` PDF를 200 OK, 58쪽 본문, 목차 기준으로 확인해 한국어 Starter Pack으로 승격했다.

3. 모든 페르소나의 상위 8개 추천 자료는 볼 단원/섹션 누락이 없었다.
   - 조치: 기본 `recommendedSections` fallback과 주요 검토 자료의 직접 섹션을 유지한다.

4. P3, P4, P6은 한국어 직접 자료가 약했으나 P6 데이터센터 전력은 추가 보강했다.
   - 조치: P3은 KOCW `반도체 공정` 강좌에서 리소그래피, 건식각, 습식각과 세정, CMOS 종합공정을 볼 단원으로 확정하고 `reviewed`로 승격했다.
   - 조치: P4는 KOCW `전력전자시스템 해석` 강좌에서 스위치 소자, PWM 컨버터, 3상 인버터, PSIM, BLDC 드라이버 실습 단원을 확정하고 `reviewed` Starter Pack으로 승격했다.
   - 조치: P6은 Smartspace `데이터센터 전기설비 구성 및 설계 가이드`와 한국전기기술인협회 `자가용전기설비 운전 및 유지관리` 훈련 목표를 확인해 `reviewed` Starter Pack으로 추가했다.
   - 잔여: P6 상위 8개 중 8위는 여전히 GSEEK candidate라 보조 후보 수준이다. 실제 면접 준비의 핵심 자료로는 상위 1-7위를 우선 사용한다.

5. 전공-직무 매칭은 과대평가를 줄이기 위해 3단계로 재보정했다.
   - 조치: 전공별 세부 직무를 `전공 직결`, `전공 확장`, `도전 직무`로 분리했다. 전체 산업 보기에서는 도전 직무를 숨기고, 산업/검색/직무군 필터를 좁혔을 때만 노출한다.
   - 조치: 산업 필터는 트랙 태그가 아니라 세부 직무의 산업 태그를 기준으로 통과시킨다. 이로써 `AI` 필터에서 자동차·화학 기본 직무가 상단에 새어 들어오던 문제를 제거했다.

6. 직무-교육 매칭은 후보 자료가 검수 자료를 앞지르지 않도록 보정했다.
   - 조치: `candidate`, `reviewNeeded`, `broad` 자료에 점수 패널티를 추가했다. 후보 자료는 보조 후보로 남기되 Starter Pack과 상위 추천은 reviewed/verified 자료를 우선한다.
   - 조치: CAE 해석 엔지니어의 기초 재료역학 자료인 MIT Mechanics를 직무 직접 연결 자료로 보정했다.
   - 조치: 추천 메인 정렬에서 reviewed/verified 비후보 자료를 후보 자료보다 우선하도록 했다. P1/P2/P3/P4/P5는 상위 8개가 모두 reviewed/verified이며, P6은 검수 자료가 7개라 8번째에만 GSEEK 보조 후보가 남는다.
   - 조치: 주차별 상세 추천과 실습 후보 정렬에도 같은 신뢰도 기준을 적용했다. 브라우저 QA 기준 P1/P3/P6 desktop/mobile 화면의 상세 추천 상위 4개에는 후보 자료가 남지 않았다.
   - 주의: P1/P2의 한국어 broad 후보는 상위 8개에서는 빠졌지만 참고자료 후보 풀에는 남아 있다. 검수 전까지 핵심 추천으로 보지 않는다.

7. 추가 재감사에서 P5 예지보전 AI의 하위 추천 정합성을 보정했다.
   - 발견: AI 산업 공통 진단에 자율주행 전용 `자율주행 AI` 항목이 섞여 예지보전에도 ROS/Jetson 성격 자료가 일부 올라왔다.
   - 조치: 공통 진단을 `AI 적용 기준`으로 바꾸고, ROS Toolbox는 로봇·자동화 트랙 전용으로 제한했다.
   - 조치: 예지보전 AI의 직접 교육 연결에 Statistics Onramp, Coursera Engineering Data, freeCodeCamp Python·데이터 프로젝트를 추가했다.
   - 결과: P5 선택 직무 순위는 5위에서 1위로 개선됐고, 상위 추천은 Predictive Maintenance, Google ML, Machine Learning Onramp, Statistics, Coursera Data, Signal Processing, Python 데이터 분석 중심으로 정렬된다.

8. 브라우저 QA에서 P1/P3/P5 대표 흐름을 desktop/mobile로 재확인했다.
   - 확인: P1 CAE와 P3 반도체 식각은 `전공 직결`, P5 예지보전 AI는 `전공 확장`으로 표시된다.
   - 확인: P1 상위 추천은 FEA, Ansys, Mechanics 중심이고 P3 상위 추천은 Semiconductor Devices, KOCW 반도체 공정, Statistics 중심이다.
   - 확인: P5 상위 추천은 Predictive Maintenance, Google ML, Machine Learning Onramp, Coursera Data, Statistics, freeCodeCamp, Signal Processing 중심으로 정렬된다.
   - 조치: P5 진단 가이드에서 AI 프로필 `keywords`가 없을 때 화면이 중단되는 오류를 수정했다.
   - 결과: P1/P3/P5 desktop 1366x900, mobile 390x844에서 후보 자료 상위 노출 0건, 가로 오버플로 0건을 확인했다. P5에서는 ROS Toolbox와 NVIDIA Jetson이 노출되지 않는다.
