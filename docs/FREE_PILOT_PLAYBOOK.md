# 무료 파일럿 운영안

## 목표

2~4주 동안 기계공학, 전자공학, 화학공학 학생 20~30명에게 직무역량 로드맵을 사용하게 하고, 다음 5가지를 검증한다.

1. 채용공고 수준 세부 직무 선택이 직무 이해도를 높이는가
2. 직무별 워드클라우드와 역량 체크가 학습 우선순위를 정하는 데 도움이 되는가
3. 주차별 공개자료 자동 배치가 무작위 검색보다 유용한가
4. 2/4/8/12주 로드맵이 준비 기간에 맞게 실행 가능한가
5. 내 커리큘럼과 엑셀 내보내기가 상담/면접 준비에 도움이 되는가

## 무료 운영 구조

| 영역 | 도구 | 비용 |
|---|---|---|
| 배포 | Cloudflare Pages | 무료 |
| 저장소 | GitHub public repository | 무료 |
| 앱 형태 | PWA | 무료 |
| 사용자 저장 | 브라우저 localStorage | 무료 |
| 설문 | Google Forms 또는 Microsoft Forms | 무료 |
| 자료 | 공개자료 원본 링크 | 무료 |

앱스토어 배포는 파일럿 범위에서 제외한다. iOS 개발자 계정은 연 99달러, Google Play 개발자 계정은 25달러 비용이 있고 심사와 유지보수 시간이 필요하다. 파일럿에서는 PWA로 반복 사용 의사를 먼저 검증한다.

## 참여자 운영

| 항목 | 기준 |
|---|---|
| 대상 | 기계공학/전자공학/화학공학 취준생 |
| 규모 | 20~30명으로 시작 |
| 기간 | 2주 빠른 파일럿 또는 4주 정식 파일럿 |
| 참여 방식 | URL 접속, 세부 직무 선택, 역량 체크, 교육 선택, 내 커리큘럼 확인/내보내기 |
| 보상 | 가능하면 커피쿠폰 등 소액. 무료 운영이면 학습 리포트 제공으로 대체 |

## 사전 설문 문항

1. 전공은 무엇인가
2. 관심 산업은 무엇인가
3. 지원하려는 세부 직무를 얼마나 명확히 알고 있는가
4. 준비 기간은 어느 정도인가
5. 취업 준비를 위해 최근 2주 안에 찾아본 자료는 무엇인가

## 사용 후 설문 문항

1. 선택한 세부 직무의 실제 업무가 더 명확해졌는가
2. 워드클라우드가 직무 핵심 역량을 이해하는 데 도움이 되었는가
3. 역량 체크 후 무엇부터 공부해야 할지 판단하기 쉬워졌는가
4. 주차별 추천 교육자료가 직접 검색보다 유용했는가
5. 선택한 준비 기간의 로드맵이 실행 가능한 수준이었는가
6. 내 커리큘럼 또는 엑셀 내보내기가 상담/면접 준비에 도움이 되었는가
7. 가장 부족하거나 부정확했던 내용은 무엇인가

## 측정 지표

| 지표 | 목표 |
|---|---|
| 세부 직무 선택률 | 80% 이상 |
| 역량 체크 완료율 | 60% 이상 |
| 교육자료 내 커리큘럼 추가율 | 40% 이상 |
| 내 커리큘럼 또는 엑셀 내보내기 사용률 | 30% 이상 |
| 1주 후 재방문 의향 | 50% 이상 |

정적 앱은 서버 로그를 수집하지 않으므로, 초기에는 설문과 학생이 내보낸 커리큘럼 파일 또는 화면 공유를 기준으로 측정한다. 필요성이 확인되면 Cloudflare Web Analytics 또는 Supabase 저장을 추가한다.

## 다음 개선 우선순위

1. 실제 학생 피드백으로 직무 설명과 역량 체크 수정
2. 자료 URL의 최신성, 라이선스, 난이도 검수
3. 직무군별 핵심 공개자료의 단원/섹션 지정
4. 내 커리큘럼 결과를 Google Forms나 Supabase로 제출하는 기능 추가
5. 반복 사용이 확인되면 네이티브 앱 또는 React Native/Expo 검토
