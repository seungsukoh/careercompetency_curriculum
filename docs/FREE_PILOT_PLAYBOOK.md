# 무료 파일럿 운영안

## 목표

4주 동안 기계공학/전자공학 학생 30명 안팎에게 직무역량 로드맵을 사용하게 하고, 다음 5가지를 검증한다.

1. 직무별 실제 업무 설명이 직무 이해도를 높이는가
2. 역량 체크리스트가 학습 우선순위를 정하는 데 도움이 되는가
3. 공개자료 큐레이션이 무작위 검색보다 유용한가
4. 4주 로드맵이 실제 학습 행동으로 이어지는가
5. 산출물 안내가 포트폴리오 작성에 도움이 되는가

## 무료 운영 구조

| 영역 | 도구 | 비용 |
|---|---|---|
| 배포 | Cloudflare Pages | 무료 |
| 저장소 | GitHub public repository | 무료 |
| 앱 형태 | PWA | 무료 |
| 사용자 저장 | 브라우저 localStorage | 무료 |
| 설문 | Google Forms 또는 Microsoft Forms | 무료 |
| 자료 | 공개자료 원본 링크 | 무료 |

앱스토어 배포는 파일럿 범위에서 제외한다. iOS 개발자 계정은 연 99달러, Google Play 개발자 계정은 25달러 비용이 있고 심사와 유지보수 시간이 필요하다. 파일럿에서는 PWA로 반복 사용 의사를 먼저 검증한다.

## 참여자 운영

| 항목 | 기준 |
|---|---|
| 대상 | 기계공학/전자공학 2~4학년 |
| 규모 | 20~30명으로 시작 |
| 기간 | 2주 빠른 파일럿 또는 4주 정식 파일럿 |
| 참여 방식 | URL 접속, 직무 트랙 선택, 역량 진단, 자료 저장, 로드맵 수행 |
| 보상 | 가능하면 커피쿠폰 등 소액. 무료 운영이면 학습 리포트 제공으로 대체 |

## 사전 설문 문항

1. 전공과 학년은 무엇인가
2. 관심 산업은 무엇인가
3. 관심 직무를 얼마나 명확히 알고 있는가
4. 현재 가장 막히는 부분은 무엇인가
5. 취업 준비를 위해 최근 2주 안에 찾아본 자료는 무엇인가

## 사용 후 설문 문항

1. 선택한 직무의 실제 업무가 더 명확해졌는가
2. 무엇부터 공부해야 할지 판단하기 쉬워졌는가
3. 추천 자료가 직접 검색보다 유용했는가
4. 4주 로드맵이 실행 가능한 수준이었는가
5. 포트폴리오로 남길 산출물이 분명했는가
6. 다시 사용할 의향이 있는가
7. 가장 부족하거나 부정확했던 내용은 무엇인가

## 측정 지표

| 지표 | 목표 |
|---|---|
| 직무 트랙 선택률 | 80% 이상 |
| 역량 진단 완료율 | 60% 이상 |
| 자료 저장률 | 40% 이상 |
| 파일럿 결과 복사 사용률 | 30% 이상 |
| 1주 후 재방문 의향 | 50% 이상 |

정적 앱은 서버 로그를 수집하지 않으므로, 초기에는 설문과 학생이 복사한 파일럿 결과를 기준으로 측정한다. 필요성이 확인되면 Cloudflare Web Analytics 또는 Supabase 저장을 추가한다.

## 다음 개선 우선순위

1. 실제 학생 피드백으로 직무 설명과 체크리스트 수정
2. 자료 URL의 최신성, 라이선스, 난이도 검수
3. 직무 트랙별 10개 이상 공개자료 확보
4. 저장 결과를 Google Forms나 Supabase로 제출하는 기능 추가
5. 반복 사용이 확인되면 네이티브 앱 또는 React Native/Expo 검토
