# QA 체크리스트

## 사용 원칙

이 체크리스트는 현재 앱 흐름인 `직무 선택 -> 워드클라우드 -> 역량 체크 -> 교육 선택 -> 내 커리큘럼 내보내기`를 검증한다.

검증 환경:

- 개발 서버: `npm.cmd run dev`
- 배포 산출물 확인: `npm.cmd run build` 후 `npm.cmd run preview`
- 브라우저: Chrome 또는 Edge 최신 버전
- 화면 폭: 데스크톱 1280px 이상, 모바일 390px 안팎

자동 사전 검증:

- `node --check app.js`
- `node --check data/roleExpansions.js`
- `npm.cmd run validate:data`
- `npm.cmd run check`

## 기능 검증

| ID | 요구사항 | 확인 절차 | 기대 결과 | 결과 |
|---|---|---|---|---|
| R1 | 세부 직무 선택 | 전공/산업/검색 필터를 바꾸고 채용공고 수준 직무를 선택한다. | 선택 직무가 강조되고 바로 아래 워드클라우드가 보인다. | 미검증 |
| R2 | 직무 상세 | 선택 직무 상세 영역을 확인한다. | 반복 업무, 자격요건, 우대 역량, 확인 검색어가 보인다. | 미검증 |
| R3 | 역량 체크 | 체크박스를 선택/해제한다. | 역량 확보율, 보완 역량, 로드맵 추천이 즉시 갱신된다. | 통과 - P6 0% -> 8% 갱신 |
| R4 | 교육 선택 | 준비 기간을 2/4/8/12주로 바꾼다. | 주차 수, 과제 반복/보강 방식, 주차별 교육자료 수가 달라진다. | 미검증 |
| R5 | 학습목표 반영 | 직무 탐색/전공 보완/포트폴리오/면접 준비를 바꾼다. | 교육 선택 화면의 활성 기준과 과제 우선순위가 바뀐다. | 미검증 |
| R6 | 주차별 교육자료 | 로드맵 주차의 추천 교육자료를 펼친다. | 자료 형식, 언어, 볼 단원/섹션, 자동추천 점수, 배치 근거, 산출물, 원본 링크가 보인다. | 미검증 |
| R7 | 내 커리큘럼 추가 | 주차 안의 교육자료에서 `내 커리큘럼에 추가`를 누른다. | 내 커리큘럼 화면의 완성 로드맵에 추가 자료가 우선 반영된다. | 통과 - Smartspace 자료 추가 확인 |
| R8 | 주차 완료 | 내 커리큘럼에서 주차 완료를 체크하고 새로고침한다. | 완료 상태가 유지된다. | 통과 - localStorage 유지, 내 커리큘럼 탭에서 복원 확인 |
| R9 | 엑셀 내보내기 | 내 커리큘럼에서 엑셀 내보내기를 실행한다. | `요약`, `실행계획`, `선택직무상세`, `회사공고대조`, `교육자료`, `역량체크` 시트가 포함된 `.xls` 파일이 내려받아진다. | 부분 통과 - Blob/파일명/시트 검증, headless 실제 파일 저장은 미확인 |
| R10 | 정적 PWA | 새로고침하고 주요 탭을 이동한다. | 서버 API 없이 앱이 동작한다. | 미검증 |

| Q2 | MathWorks 매핑 | CAE, 제어, 품질/수율, 전원/모터 관련 직무를 선택한다. | 필요한 경우 MATLAB/Simulink/Simscape/Statistics 자료가 상위 추천에 포함된다. | 미검증 |
| Q3 | Coursera/글로벌 자료 | 화학공정, 품질, 임베디드 직무를 확인한다. | Coursera, edX, Khan Academy, freeCodeCamp 후보가 적절한 주차에 연결된다. | 미검증 |
| Q4 | 한국어 접근성 | KOCW, K-MOOC, HRD-Net, GSEEK 후보를 확인한다. | 입문 또는 전공 보완 목표에서 한국어 자료가 충분히 노출된다. | 미검증 |
| Q5 | 워드클라우드 | 모든 직무군에서 대표 직무를 선택한다. | 직무 키워드 기반 동적 워드클라우드가 보이고 모바일 폭에서도 넘치지 않는다. | 부분 통과 - P6 데스크톱/모바일 확인 |
| Q6 | 필수 Starter Pack | core 자료가 있는 세부 직무를 선택한다. | 교육 선택 화면에 필수 Starter Pack 영역이 보이고 먼저 완료할 자료로 안내된다. candidate 또는 reviewNeeded 자료는 포함되지 않는다. | 통과 - P6 Edge/Chrome headless |
| Q7 | 후보 승격 기준 | reviewed로 승격된 자료 카드를 확인한다. | 플랫폼 전체 링크가 아니라 볼 단원/섹션이 지정되어 있고, 최종 URL이 실제 제목·본문·파일명 기준으로 열리며, 검색 페이지·차단 링크 후보는 candidate로 남아 있다. | 부분 통과 - P6 Smartspace/한국전기기술인협회 확인 |
